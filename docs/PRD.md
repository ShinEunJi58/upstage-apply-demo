## 1. 프로젝트 개요 (Executive Summary)
**NSR Smart Navigator**는 복잡한 북극항로(NSR: Northern Sea Route) 물류 데이터를 AI 에이전트가 실시간으로 분석하여, 유럽 수출을 희망하는 중소기업에게 최적의 물류 경로와 리스크 정보를 대화형으로 제공하는 서비스입니다. 
KOTRA의 핵심 업무인 **'시장 정보 조사'**와 **'해외진출 지원'** 업무의 디지털 전환(DX) 예시를 시각적으로 구현하여, 무역 업무의 효율성을 극대화하는 것을 목표로 합니다.

## 2. 기획 배경 및 문제 정의 (Background & Problem)
### 2.1 현황
수에즈 운하 사태 등 글로벌 물류 불안정으로 인해 **북극항로(NSR)**가 대안으로 부상하고 있으나, 중소기업은 정보 부족과 지정학적 리스크(러시아-우크라이나 이슈 등)로 인해 접근이 어려운 상황입니다.

### 2.2 문제점 (Pain Points)
1.  **정보 비대칭**: 북극항로 운임, 해빙 정보, 운항 가능 여부 등을 확인하기 위해 여러 사이트를 개별적으로 탐색해야 함.
2.  **의사결정의 어려움**: 자사 제품이 북극항로를 이용하기에 적합한지(운송 시간, 비용 대비 효율 등) 판단할 기준이 모호함.
3.  **전문가 상담의 한계**: KOTRA 무역관 인력이 모든 개별 기업의 단순 반복 문의를 실시간으로 응대하기 어려움.

## 3. 타겟 유저 (Target Audience)
*   **Primary User**: 유럽 수출을 희망하는 국내 중소·중견기업 물류 담당자. (비용 절감과 빠른 배송이 필요한 실무자)
*   **Secondary User**: KOTRA 해외무역관 직원. (기업 상담 시 기초 데이터 제공 및 보조 도구로 활용)

## 4. 해결책 및 핵심 가치 (Solution & Value Prop)
POPOW 기반의 AI 에이전트가 뉴스, 운임, 품목 데이터를 학습하여 24시간 무역 상담을 수행합니다.

*   **실시간 데이터 분석**: 뉴스, 해빙 농도 등 변동성 데이터를 실시간으로 반영.
*   **맞춤형 컨설팅**: 사용자의 화물 품목과 목적지에 따른 최적 경로 제안.
*   **무역 업무 DX**: 단순 정보 조사는 AI에게 맡기고, 사람은 고부가가치 의사결정에 집중.

## 5. 핵심 기능 (Key Features)

### 5.1 북극항로 경제성 진단 (NSR Cost Calculator)
*   **기능 설명**: 기존 항로(수에즈) 대비 북극항로 이용 시의 거리, 예상 소요 시간(Lead Time), 연료 절감액 비교 분석 제공.
*   **사용자 입력 예시**: "부산에서 함부르크까지 컨테이너 5개 보내는데 북극항로 어때?"
*   **구현 방식**: '운임 비교 엑셀 데이터(Knowledge Base)' 연동.

### 5.2 지정학적 리스크 모니터링 (Risk Radar)
*   **기능 설명**: 최근 러시아 제재 뉴스, 북극 해빙 농도 데이터 등을 종합하여 [안전 / 주의 / 위험] 3단계 신호등 형태로 리스크 브리핑.
*   **사용자 입력 예시**: "지금 북극항로 이용해도 안전해?"
*   **구현 방식**: POPOW Web Browsing 플러그인 활용 실시간 뉴스 검색.

### 5.3 수출 유망 품목 추천 (Item Matcher)
*   **기능 설명**: 배송 속도가 중요한 품목(신선식품, 패스트패션 등)을 추천하고, 해당 품목의 유럽 내 HS Code 및 수출 팁 안내.
*   **사용자 입력 예시**: "북극항로로 뭘 팔면 좋을까?"
*   **구현 방식**: 'K-Food/K-Beauty 유망 품목 리스트' Knowledge Base 학습.

## 6. 에이전트 워크플로우 (User Flow)
1.  **Start**: 유저가 POPOW 채팅창 접속.
2.  **Input**: 자연어 질의 입력 (예: "유럽으로 화장품 수출하고 싶은데 물류비가 걱정이야.")
3.  **Reasoning (AI)**:
    *   화장품 → 트렌드 민감 → 빠른 배송 필요.
    *   물류비 절감 니즈 → 북극항로 대안 검토.
4.  **Action**:
    *   *Knowledge Base 검색*: 화장품의 북극항로 적합도 확인.
    *   *Web Browsing*: 현재 운항 가능 여부 및 리스크 확인.
5.  **Output**: 종합 솔루션 제시.
    *   "화장품은 유행이 빨라 북극항로(25일)가 수에즈(40일)보다 유리합니다. 단, 현재 러시아 해역 이슈로 '주의' 단계이니 보험 가입을 권장합니다. 예상 절감액은 약 15%입니다."

## 7. 기대 효과 (Success Metrics)
*   **정량적**: 기업 상담 시간 단축 (3시간 → 3분), 중소기업 북극항로 문의 응대율 50% 증가.
*   **정성적**: KOTRA의 디지털 혁신 이미지 제고, 직원의 고부가가치 업무 집중도 향상.

## 8. 기술 스택 (Tech Stack)
*   **AI Engine**: POPOW.ai (AI Agent Builder)
*   **Knowledge Base**: 운임 데이터, 국가별 HS Code, 유망 품목 리스트 (Excel/PDF)
*   **Tools**: Web Browsing Plugin (실시간 정보 수집)
