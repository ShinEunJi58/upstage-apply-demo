[PRD] KOTRA 통상직 직무 특화: POPOW 기반 NSR(북극항로) 수출 전략 에이전트
1. 프로젝트 개요 (Executive Summary)
프로젝트명: NSR Smart Navigator (북극항로 수출 전략 에이전트)

기획자: [지원자 성명] (KOTRA 통상직 지원자)

구현 도구: POPOW.ai (AI Agent Builder)

목적: 복잡한 북극항로(NSR) 물류 데이터를 AI 에이전트가 실시간으로 분석하여, 중소기업에게 최적의 물류 경로와 리스크 정보를 대화형으로 제공함.

핵심 가치: KOTRA의 핵심 업무인 **'시장 정보 조사'**와 '해외진출 지원' 업무의 디지털 전환(DX) 예시를 시각적으로 구현.

2. 기획 배경 및 문제 정의 (Why & Problem)
현황: 수에즈 운하 사태 등 글로벌 물류 불안정으로 인해 북극항로가 대안으로 떠오르고 있으나, 중소기업은 정보 부족과 러시아 지정학적 리스크로 인해 접근이 어려움.

문제점 (Pain Point):

북극항로 운임 및 해빙 정보를 얻기 위해 여러 사이트를 뒤져야 함 (정보 비대칭).

우리 제품이 북극항로에 적합한지 판단할 기준이 모호함 (의사결정 어려움).

해결책 (Solution): POPOW 기반의 AI 에이전트가 뉴스, 운임, 품목 데이터를 학습하여 24시간 무역 상담을 수행.

3. 타겟 유저 및 페르소나 (Target User)
Primary User: 유럽 수출을 희망하는 국내 중소·중견기업 물류 담당자.

Secondary User: KOTRA 해외무역관 직원 (기업 상담 시 보조 도구로 활용).

4. 핵심 기능 및 JD 키워드 매칭 (Key Features)
POPOW 에이전트가 수행할 3가지 핵심 기능(Skill)을 KOTRA JD 키워드와 연결했습니다.

기능 1. 북극항로 경제성 진단 (NSR Cost Calculator)
JD 키워드: 자원관리능력, 전문지식과 그 응용능력

기능 상세:

사용자가 "부산에서 함부르크까지 컨테이너 5개 보내는데 북극항로 어때?"라고 물으면,

**[수에즈 vs 북극항로]**의 거리, 예상 소요 시간(Lead Time), 연료 절감액을 비교하여 답변.

POPOW 구현 방식:

미리 조사한 '운임 비교 엑셀 데이터(Knowledge Base)'를 POPOW에 업로드하여 에이전트가 참조하도록 설정.

기능 2. 지정학적 리스크 모니터링 (Risk Radar)
JD 키워드: 시장 정보 조사, 정보능력, 내부 경영관리

기능 상세:

"지금 북극항로 이용해도 안전해?"라고 물으면,

최근 1주일간의 러시아 제재 뉴스, 북극 해빙 농도 데이터를 요약하여 [안전 / 주의 / 위험] 신호등 형태로 답변.

POPOW 구현 방식:

POPOW의 'Web Browsing' 플러그인을 활성화하여 실시간 뉴스 검색 기능 연동.

기능 3. 수출 유망 품목 추천 (Item Matcher)
JD 키워드: 국내기업을 위한 해외시장개척, 창의력

기능 상세:

"북극항로로 뭘 팔면 좋을까?"라고 물으면,

'신선식품(냉동만두)', '패스트 패션' 등 빠른 배송이 생명인 품목을 추천하고, 해당 품목의 유럽 내 HS Code를 안내.

POPOW 구현 방식:

'K-Food/K-Beauty 유망 품목 리스트' 문서를 POPOW Knowledge에 학습시킴.

5. 에이전트 워크플로우 (User Flow on POPOW)
이 흐름도를 포트폴리오에 넣으면 **"업무 프로세스 설계 능력"**이 돋보입니다.

Start: 유저가 POPOW 채팅창에 접속.

Input: "유럽으로 화장품을 보내려는데 물류비가 너무 비싸요." (자연어 입력)

Reasoning (AI 사고):

화장품(품목) → 유행에 민감함 → 빠른 배송 필요.

물류비 절감 니즈 → 북극항로 대안 검토 필요.

Action (도구 호출):

Knowledge Base 검색: 화장품의 북극항로 적합도 확인.

Web Browsing: 현재 북극항로 운항 가능 여부 확인.

Output (답변):

"화장품은 유행이 빨라 북극항로(25일 소요)가 수에즈(40일)보다 유리합니다. 단, 현재 러시아 해역 이슈가 있으니 '주의' 단계입니다. 예상 절감 비용은 약 15%입니다."

6. 기대 효과 (Success Metrics)
정량적 효과:

기업 상담 시간 단축: 기존 자료 검색 3시간 → 에이전트 질의 3분.

정보 접근성 향상: 중소기업의 북극항로 문의 응대율 50% 증가 예상.

정성적 효과:

KOTRA의 '디지털 무역 지원' 이미지를 강화하고, 단순 반복 상담 업무를 AI에게 일임하여 직원은 고부가가치 업무에 집중.

7. 포트폴리오 제출 가이드 (How to present)
1) POPOW 에이전트 링크 제출:

실제로 구동되는 에이전트 링크를 자소서나 이력서에 QR코드로 첨부하세요.

멘트: "제가 만든 AI 무역 사원입니다. 한번 대화해보세요."

2) 스크린샷 활용:

에이전트가 "북극항로 리스크 분석 결과입니다"라고 똑똑하게 대답하는 화면을 캡처하여 포트폴리오 문서에 넣으세요.

3) KOTRA 면접 멘트:

"요즘 중소기업들은 정보를 찾는 시간조차 부족합니다. 저는 코딩은 못하지만, POPOW 같은 AI 도구를 활용해 KOTRA의 무역 데이터를 **'살아있는 상담원'**으로 만드는 방법을 압니다. 이 **[NSR 스마트 네비게이터]**가 그 증거입니다."

이 PRD대로만 만드시면, **"개발자도 아닌데 최신 AI 툴로 무역 업무를 혁신하려는 스마트한 인재"**라는 평가를 확실히 받으실 수 있습니다.
